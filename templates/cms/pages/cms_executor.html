EContest Executor

EContest Executor - это скрипт для управления некоторыми административными действиями в EContest через терминал.

Правила запуска

Для того, чтобы использовать команду, перейдите в директорию с EContest.
Затем, напишите python executor.py ... (пишите python/py в зависимости от версии и от названия в PATH)

Доступные команды

Команда и аргументы Что делает команда
set-admin [user] Сделать пользователя с логином user администратором
remove-admin [user] Убрать пользователя с логином user из администраторов
check-user [user] Проверить, существует ли пользователь с логином user
remove-user [user] Удалить пользователя с логином user
ban-user [user] Забанить пользователя с логином user
unban-user [user] Разбанить пользователя с логином user
userlist Вывести список всех пользователей с информацией о них
